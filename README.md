# Image-Processing-and-Computer-Vision---YOLO-Implementation-for-traffic-sign-recognition
This is an Image Processing and Computer Vision project that I did during the second semester (Spring 2026) of my Master's degree. This project deals with the implementation of the You Only Look Once (YOLO) model to detect traffic signs

**Instructions on how to run the project:**
If you don't have Google Colab Pro, then it's best to run the notebook and all of its cells on Kaggle since it contains a free trial GPU which speeds up the YOLO training process.
